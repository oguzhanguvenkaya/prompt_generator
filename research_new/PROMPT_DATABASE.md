### Model Selection Logic

| User Intent | Recommended Model | Reason |
|-------------|-------------------|--------|
| Video with sound/music | Veo 3.1 | Only model with native audio generation |
| Anime/stylized video | Kling 3.0 | Best anime support + creativity parameter |
| Multi-shot video sequence | Seedance 2.0 | Cut to: syntax + @reference system |
| Product photography | Seedream 2.0 / Nano Banana Pro | Material detail + lighting precision |
| Brand-consistent imagery | Recraft Image | Style presets + exact color palette |
| Icon / logo / vector | Recraft Vector | Native SVG output, clean lines |
| Fine art / editorial photo | Flux | Natural language, photorealism |
| Complex scene with references | Nano Banana Pro | 14 reference images + web search |

### Prompt Pattern Analysis

#### High-Success Patterns (Score 9-10)

1. **Specificity over vagueness**: 'brushed aluminum surface under 45-degree side lighting' > 'metallic product'
2. **Named techniques**: 'Rembrandt lighting' > 'dramatic lighting'
3. **Camera specification**: 'Canon EOS R5 with 85mm f/1.4 at f/2.0' > 'professional camera'
4. **Scene narrative**: Describe the complete scene, not just the subject
5. **Model-specific syntax**: Use each model's unique features (timestamps, @references, creativity param)

#### Common Failure Patterns (Avoid)

1. **Keyword lists for Flux**: Never use comma-separated keywords; use full sentences
2. **Missing audio for Veo 3.1**: Always include Audio: section for video prompts
3. **No creativity param for Kling**: Specify creativity=0.5/0.7/1.0 for predictable results
4. **Generic lighting**: 'good lighting' fails; 'soft diffused natural light from the left' succeeds
5. **Missing material specs for Seedream**: Always specify surface material and texture
